# Help Desk System

## Overview
This project is a **Help Desk System** developed to simulate a real-world corporate environment.  
Its main goal is to consolidate backend development skills, database integration, and business rule implementation.

The system allows users to open and track support tickets, while managers and administrators can manage users and monitor tickets by department.

---

## Features
- Ticket creation with department assignment  
- Ticket tracking and detailed view  
- Role-based access control:
  - Administrator
  - Manager
  - User
- User management
- Full ticket lifecycle:
  - Open
  - View
  - Close

---

## Technologies Used
- PHP (Object-Oriented Programming)
- MySQL
- HTML5
- CSS3
- Bootstrap

---

## System Roles
- **User**: Can open and track their own tickets  
- **Manager**: Can view and manage tickets from their department  
- **Administrator**: Full system access, including user management

---

This project was created for learning and practice purposes, focusing on:
