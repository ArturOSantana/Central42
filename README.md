# HelpDeks
HelpDesk em PHP - Repositório Criado para Projeto de PHP nas aulas de Desenvolvimento de Sistemas
<hr>



# FUNCIONALIDADES: 
Serviço para abrir chamado de suporte, organização de chamado para os departamentos responsaveis. 
Cada chamado aberto deve ter um tempo util.
Cada usuario deve conseguir visualizar apenas os seus chamados, e os administradores devem conseguir visualizar apenas os sistemas do seu departamento.
Após cada chamado deve se ter um feedback para que o usuario possa avaliar.
